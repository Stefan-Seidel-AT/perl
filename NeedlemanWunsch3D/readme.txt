# Readme

This program uses the Needleman-Wunsch algorithm for 3 sequences.
It takes 3 sequences in fasta-format in one file, aligns them and prints
the output to the console.

# usage (with perl installed):
   ./NeedlemanWunsch3D.pl name_of_seq_file

for more detailed and general information about the Algorithm please look here:
https://en.wikipedia.org/wiki/Needleman%E2%80%93Wunsch_algorithm
